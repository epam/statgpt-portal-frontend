# DIAL Toolkit

The DIAL Toolkit provides utilities and helpers for working with DIAL APIs in the StatGPT ecosystem.

## Features

- Utility functions for DIAL data processing
- Type definitions for DIAL concepts
- Helpers for integration with DIAL APIs

## Installation

```bash
npm install @statgpt/dial-toolkit
```

## Usage

```typescript
import { someDialUtility, DialType } from '@statgpt/dial-toolkit';

// Example usage
const result = someDialUtility(data);
```
