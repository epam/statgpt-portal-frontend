# Shared Toolkit

This library provides shared utilities, types, and helpers used across multiple projects in the StatGPT ecosystem.

## Features

- Common utility functions
- Shared type definitions
- Reusable helpers for data processing and formatting

## Installation

```bash
npm install @statgpt/shared-toolkit
```

## Usage

```typescript
import { getFilteredItemsWithParents, FormatNumbersType } from '@statgpt/shared-toolkit';

// Example usage
const result = getFilteredItemsWithParents(data);
```
