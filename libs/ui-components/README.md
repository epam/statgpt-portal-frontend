# UI Components

This library provides reusable React UI components for use across multiple projects in the StatGPT ecosystem.

## Features

- Consistent, accessible, and customizable UI components
- Components for forms, buttons, inputs, modals, loaders, and more
- Built with TypeScript and Tailwind CSS (or your project's styling system)
- Designed for easy integration and extension

## Installation

```bash
npm install @statgpt/ui-components
```

## Usage

```tsx
import { Button } from '@statgpt/ui-components';

// Example usage
<Button title="Click me" onClick={() => alert('Clicked!')} />;
```

## Development

- Source code: `libs/ui-components`
- To run tests:
  ```bash
  npm run test libs-ui-components
  ```
